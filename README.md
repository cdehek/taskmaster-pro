# Taskmaster-Pro

## An app that allows you to track your projects/items.
  * Events highlight yellow to show they are within 2 days of being past-due.
  * Events highlighted red indicated that they are currently past due.
  
### Made Using:
  * CSS
  * Html
  * JS
  * jQuery
  * Boostrap
